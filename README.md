# Daily Habit

A shared Bible reading companion for two accountability partners.
Built with React + Vite. Deploys as a static site (Netlify, Vercel, etc).

## Run locally
```
npm install
npm run dev
```

## Build for production
```
npm run build
```
Output goes to `dist/`. Deploy that folder, or connect the repo to Netlify
(build command `npm run build`, publish directory `dist`).

See HOW-TO-DEPLOY.txt for beginner-friendly step-by-step instructions.
